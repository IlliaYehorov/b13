Інструкція по віддаленому користуванню офісним сервером B13-QNAP-1688

Ви можете користуватись робочими файлами з нашого сервера на своєму домашньому Mac чи PC. Для цього ми використовуємо захищений VPN-вузол. З увімкненим підключенням ви працюєте так, ніби знаходитесь в локальній мережі офісу, але з обмеженнями по швидкості інтернету та шифрування.

Важливо:
  • Файли на сервері — це єдині оригінали. Видалення файлу — це видалення для всієї команди.
  • Розміщуйте проекти в папках року, назви у форматі РРРР-ММ-ДД_Назва_Автор.
  • Після роботи вимикайте VPN, щоб не перевантажувати мережу.

Надішліть цей IP Іллі для отримання конфіг-файлу.
Крок 3Підключення
  • Відкрийте WireGuard ➔ Import tunnel(s) from file.
  • Виберіть .conf файл від Іллі ➔ Activate.
Крок 4Робота з сервером

Підключення до сховища (Cmd+K):

smb://192.168.88.200
Крок 5Відключення

Після роботи натисніть Deactivate. Це звільнить канал офісної мережі та поверне швидкість вашому домашньому інтернету.

FAQМожливі проблеми

Якщо з'єднання не працює:

  • В офісі відсутні світло або інтернет.
  • Ваш домашній провайдер блокує VPN.
  • Змінилася ваша IP-адреса (динамічний IP).

Якщо проблема не зникає — напишіть Іллі.
