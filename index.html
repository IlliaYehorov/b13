Інструкція по віддаленому користуванню офісним сервером B13-QNAP-1688

Ви можете користуватись робочими файлами з нашого сервера на своєму домашньому Mac чи PC. Для цього ми використовуємо захищений VPN-вузол. З увімкненим підключенням ви працюєте так, ніби знаходитесь в локальній мережі офісу, але з обмеженнями по швидкості інтернету та шифрування. Підключення дозволить вам переглядати матеріали прямо з сервера, завантажувати туди свої матеріали чи викачувати їх для локального монтажу.

Важливо:
  • Файли, які ви бачите у папках сервера — це єдині оригінали. Якщо ви видалите файл у себе у Finder чи Провіднику, він назавжди видалиться з сервера для всієї команди. Будьте обережні.
  • Коли VPN увімкнено, увесь ваш трафік іде через офіс. Швидкість вашого інтернету падатиме в 2-3 рази за використання цього підключення. Після роботи одразу вимикайте тунель WireGuard.
Крок 1 Встановлення клієнта

Завантажте та встановіть офіційну програму WireGuard. Вона доступна для MacOS, Windows, iOS або Android і працює у фоні.

Завантажити WireGuard
Крок 2 Авторизація мережі

Сервер не пропустить вас, якщо вашої адреси немає у білому списку. Прийдіть додому чи в те місце, з якого плануєте працювати з файлами, та підключіться до Wi-Fi чи кабелю. Не використовуйте тимчасові місця перебування, публічні Wi-Fi тощо. Скопіюйте свій IP нижче:

... перевірка локації...
Надішліть цей IP Іллі та дочекайтесь від нього вашого особистого конфіг-файлу для WireGuard. Це буде ваш цифровий ключ до сервера.
Крок 3 Підключення
  • Відкрийте програму WireGuard.
  • Натисніть Import tunnel(s) from file (або Cmd + O на Mac).
  • Виберіть файл .conf, який отримали від Іллі.
  • Натисніть Activate.
Крок 4 Робота з сервером

Щоб змонтувати диск із матеріалами у Finder (для Mac):

  • У верхньому меню Finder виберіть: Перейти ➔ Під'єднатися до сервера... (або натисніть Cmd + K).
  • Вставте адресу нижче та натисніть «Під'єднатися». Після приєднання ви побачите наше основне сховище — home.
smb://192.168.88.200

Панель керування сервером (через браузер):

http://192.168.88.200:8080

Для авторизації в панелі керування потрібно встановити QNAP Authenticator: iOS або Android.

Крок 5 Відключення

Після роботи одразу вимкніть тунель WireGuard (натисніть Deactivate).

FAQ Можливі проблеми

Якщо з'єднання немає, то або в офісі зараз відсутнє світло чи інтернет, або виникли проблеми з вашим домашнім провайдером. Перевірте з'єднання пізніше, і якщо нічого не виходить — напишіть Іллі.
