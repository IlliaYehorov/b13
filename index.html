Інструкція по користуванню офісним сервером B13-QNAP-1688

Ви можете користуватись робочими файлами з нашого сервера на своєму домашньому Mac чи PC. Для цього ми використовуємо захищений VPN-вузол. Підключення дозволить вам переглядати, викачувати чи завантажувати свої матеріали напряму на сервер. Монтувати напряму з файлів на сервері не вийде через обмежену швидкість інтернету.

Важливо:
  • Файли, які ви бачите у папках сервера — це єдині оригінали. Якщо ви видалите файл у себе у Finder чи Провіднику, він назавжди видалиться з сервера для всієї команди.
  • Розміщуйте свої проекти не в головній (кореневій) папці диску, а в папках відповідного року.
  • Називайте папки своїх проектів у форматі РРРР-ММ-ДД_Назва-проекту_Автор.
  • Коли VPN увімкнено, швидкість вашого інтернету може падати. Після роботи одразу вимикайте тунель WireGuard.
Крок 1Встановлення VPN клієнта WireGuard

Завантажте та встановіть офіційну програму WireGuard. Вона доступна для MacOS, Windows, iOS або Android.

Завантажити WireGuard
Крок 2Авторизація вашого IP

Сервер не пропустить вас, якщо вашої адреси немає у білому списку. Прийдіть додому, підключіться до Wi-Fi чи по кабелю та скопіюйте свій IP нижче. Не використовуйте для цього тимчасові місця перебування чи публічні Wi-Fi.

Надішліть цей IP Іллі та дочекайтесь від нього вашого особистого конфіг-файлу для WireGuard.

Крок 3Встановлення конфігурації WireGuard
  • Відкрийте програму WireGuard.
  • Натисніть Import tunnel(s) from file.
  • Виберіть файл .conf, який отримали від Іллі.
Крок 4Підключення та робота з сервером

Натисніть Activate у WireGuard в установленому конфігу.

Скопіюйте адресу сховища:

smb://192.168.88.200

Далі у верхньому меню Finder виберіть: Перейти ➔ Під'єднатися до сервера... (або Cmd + K). Вставте адресу та виберіть том home.

Панель керування сервером через браузер:

http://192.168.88.200:8080

Для авторизації встановіть QNAP Authenticator: iOS або Android.

Крок 5Завершення роботи

Після роботи одразу вимкніть тунель WireGuard (натисніть Deactivate), щоб не навантажувати офісну мережу та повернути повну швидкість вашого інтернету.

FAQМожливі проблеми

Якщо з'єднання не відбувається:

  • В офісі немає світла або інтернету.
  • Змінилася ваша IP-адреса (динамічний IP) або ви заходите з іншого місця перебування. Ще раз перевірте свою IP адресу на цьому сайті та впевніться, що вона збігається з тою ж, яку ви подавали у білий список IP.

Якщо IP змінився — просто надішліть новий Іллі.
