Привіт, колеги.

Ви можете користуватись робочими файлами з нашого сервера на своєму домашньому Mac чи PC. Для цього ми використовуємо захищений VPN-вузол. З увімкненим підключенням ви працюєте так, ніби фізично сидите у студії. Це дозволить вам переглядати матеріали прямо з сервера, завантажувати туди свої рендери або забирати файли для локального монтажу.
УВАГА: Файли, які ви бачите у папках сервера — це єдині оригінали. Якщо ви видалите файл у себе у Finder/Провіднику, він назавжди видалиться з сервера для всієї команди. Будьте уважні!
01 Встановлення клієнта

Завантажте та встановіть офіційну програму WireGuard. Вона доступна для MacOS, Windows, iOS або Android.

Завантажити WireGuard
02 Авторизація мережі

Сервер не пропустить вас, якщо вашої адреси немає у білому списку. Визначте свій IP та надішліть його для створення доступу.

... перевірка локації...
Скопіюйте цей ІР та надішліть Іллі. У відповідь ви отримаєте файл конфігурації (.conf).
03 Підключення
  • Відкрийте програму WireGuard.
  • Натисніть Import tunnel(s) from file (або Cmd + O на Mac).
  • Виберіть файл .conf, який отримали від Іллі.
  • Натисніть Activate.
ВИМИКАЙТЕ ПІСЛЯ РОБОТИ: Коли VPN увімкнено, увесь ваш трафік іде через офіс. Обов'язково вимикайте WireGuard (Deactivate), коли закінчили роботу з файлами. Швидкість вашого інтернету падатиме в 2-3 рази за використання цього підключення, тому це у ваших інтересах його вимкнути.
04 Доступ до папок

Щоб змонтувати диск із матеріалами у Finder (для Mac):

  • У верхньому меню Finder виберіть: Перейти ➔ Під'єднатися до сервера... (або натисніть Cmd + K).
  • Вставте адресу нижче та натисніть "Під'єднатися".
smb://192.168.88.200

Панель керування сервером (через браузер):

http://192.168.88.200:8080

Можливі проблеми:

Якщо VPN увімкнений, але файли не відкриваються — можливо, ваша адреса ще не додана до білого списку. Зачекайте підтвердження від Іллі.

Якщо IP додано, але підключення все одно не працює — ймовірно, в офісі зараз відсутнє світло або інтернет, тому сервер недоступний.
